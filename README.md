# 🛡️ EmergCall
**Safety, Simplified.**

EmergCall is a quick-access safety application designed to minimize reaction time during high-stress emergencies where every second counts. This project was developed for **CS 409 - Mobile Computing** at Technological Institute of the Philippines - Quezon City (TIP-QC).

## ✨ Key Features
* **🚨 Shake-to-SOS:** Hardware-integrated panic mode that utilizes the device's accelerometer to trigger instant alerts.
* **📇 Offline Emergency Directory:** Immediate, zero-latency access to Police, Fire, and Medical hotlines without requiring an active internet connection.
* **🏥 Medical Passport (ICE):** A digital "In Case of Emergency" profile storing critical data such as blood type, allergies, and chronic conditions for first responders.
* **📍 Location & Privacy Management:** Configure location sharing with trusted contacts during an active emergency.

## 🛠️ Tech Stack
* **Frontend:** React Native 
* **Framework:** Expo
* **Navigation:** React Navigation (Native Stack)
